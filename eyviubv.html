Cenário de Phishing


O que é Phishing?

O Phishing é um tipo de ataque cibernético em que um golpista tenta enganar a vítima para que ela forneça informações sensíveis, como senhas, números de cartão de crédito ou dados bancários.


Como Evitar que Você Seja uma Vítima de Phishing

Verifique os e-mails antes de clicar em qualquer link. O e-mail geralmente contém uma mensagem alarmante ou tentadora, como:

  • "Sua conta será bloqueada em 24 horas"
  • "Você ganhou um prêmio"
  • "Atividade suspeita detectada, clique aqui para verificar"

Como Recorrer Caso Ocorram Ataques de Phishing

Caso aconteça, é preciso ser rápido, pois seus dados já podem ter sido vazados:

  • Troque as senhas de todas as suas contas online (e-mails, redes sociais, bancos). Se usou a mesma senha em diferentes sites, altere as senhas de todos eles.
  • Ative a autenticação de dois fatores (2FA) em suas contas sempre que possível. Isso adiciona uma camada extra de segurança.
  • Verifique suas transações bancárias e de cartão de crédito para identificar qualquer atividade suspeita. Informe imediatamente ao banco qualquer transação não reconhecida.

Cenário de Phishing: Descrição

Um funcionário recebe um e-mail aparentemente legítimo de seu banco, solicitando que ele clique em um link para verificar informações da conta. O link leva a uma página falsa que coleta suas credenciais.

Como os Ataques de Phishing Funcionam:

O atacante envia um e-mail que se parece com uma comunicação oficial de uma empresa, tentando induzir a vítima a clicar em um link ou baixar um anexo malicioso.

Como Identificar Sinais de um E-mail Falso

  • Verifique o remetente: O e-mail pode parecer legítimo, mas o endereço de e-mail do remetente pode ser falso ou conter pequenas variações.
  • Desconfie de links: Passe o mouse sobre links para verificar o endereço real e, se parecer estranho, não clique.
  • Erros gramaticais e de digitação: E-mails falsos frequentemente têm erros de gramática ou formatação.
  • Pedidos incomuns: Empresas legítimas não solicitam informações sensíveis, como senhas, por e-mail.

Cenário de Ransomware


O que é Ransomware?

O Ransomware é um tipo de malware que criptografa os arquivos do computador da vítima, exigindo um pagamento (resgate) para desbloquear os dados.

Como Funciona o Ataque de Ransomware:

Um usuário baixa um arquivo de um site desconhecido. O arquivo contém ransomware que criptografa todos os dados do computador e exige um pagamento para liberar os arquivos.

Como Evitar Ransomware

  • Não baixe arquivos de fontes não confiáveis. Verifique a autenticidade do site antes de fazer downloads.
  • Faça backups regulares dos seus dados. Mantenha cópias atualizadas dos arquivos importantes em locais seguros.
  • Use um antivírus atualizado para detectar e bloquear possíveis ameaças.

Cenário de Senhas Fracas


O que são Senhas Fracas?

Senhas fracas são aquelas que são fáceis de adivinhar ou são muito comuns, como "123456" ou "password".

Como Funciona o Ataque com Senhas Fracas:

Um sistema tem várias contas de usuário com senhas simples. Um atacante utiliza um ataque de força bruta para tentar adivinhar as senhas e acessar as contas.

Como Proteger Suas Contas com Senhas Fortes:

  • Use senhas longas e complexas, com uma combinação de letras, números e caracteres especiais.
  • Evite usar informações pessoais, como datas de nascimento ou nomes de familiares.
  • Utilize um gerenciador de senhas para gerar e armazenar senhas seguras.

Cenário de Software Desatualizado


O que é Software Desatualizado?

Software desatualizado é aquele que não recebe mais atualizações de segurança, o que pode deixar o sistema vulnerável a ataques.

Como Funciona o Ataque em Software Desatualizado:

Um servidor está executando uma versão antiga de um software que possui vulnerabilidades conhecidas. Um atacante explora essas vulnerabilidades para obter acesso ao servidor.

Como Proteger Seu Sistema

  • Mantenha todos os seus softwares atualizados, incluindo sistemas operacionais, navegadores e aplicativos.
  • Ative atualizações automáticas para garantir que seu sistema esteja sempre protegido contra novas ameaças.

Cenário de Pretexting


O que é Pretexting?

Pretexting é uma técnica de engenharia social onde um atacante se faz passar por uma pessoa ou instituição confiável para obter informações sensíveis.

Como Funciona o Ataque de Pretexting:

Um atacante liga para um funcionário fingindo ser do departamento de TI e solicita informações de login para "resolver um problema urgente". O funcionário, acreditando na urgência, fornece as informações.

Como Evitar Pretexting

  • Verifique sempre a identidade de quem está solicitando informações, especialmente se a solicitação parecer urgente.
  • Entre em contato diretamente com a empresa usando números ou e-mails oficiais para confirmar a solicitação.

Cenário de Baiting


O que é Baiting?

Baiting é um tipo de ataque onde um atacante deixa um dispositivo infectado, como um pendrive, em um local público, esperando que alguém o conecte a um computador.

Como Funciona o Ataque de Baiting:

Um funcionário encontra um pendrive em um local público e o conecta ao seu computador, infectando o sistema com malware.

Como Evitar Baiting

  • Não conecte dispositivos desconhecidos a seus computadores ou redes.
  • Implemente políticas de segurança rigorosas para evitar que funcionários conectem dispositivos não autorizados.

Cenário de Violação de Dados


O que é uma Violação de Dados?

Uma violação de dados ocorre quando dados confidenciais são acessados ou divulgados de maneira não autorizada.

Como Funciona a Violação de Dados:

Uma empresa descobre que dados confidenciais de clientes foram acessados por um atacante. A equipe de resposta deve identificar a origem da violação, conter o ataque, erradicar a ameaça e recuperar os dados.

Como Responder a uma Violação de Dados

  • Identifique a origem da violação e contenha o ataque o mais rápido possível.
  • Erradique a ameaça e implemente medidas para evitar futuras violações.
  • Informe as partes afetadas e, se necessário, ofereça suporte para mitigar os danos.
